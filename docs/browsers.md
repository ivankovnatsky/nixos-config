# Browsers

## Safari

Pros:

- Native on macOS
- Stable over long sessions with many tabs

Cons:

- Starting to eat up CPU time big time consuming at least 1 Core for 'Safari Networking'

## Firefox

Pros:

Cons:

- Occasional performance hiccups and hangs on macOS during long sessions, especially under low memory

## Chrome

Pros:

- Very fast

Cons:

- Heavier footprint

## Chromium

Pros:

- Lighter alternative to Chrome

Cons:

- Similar footprint issues as Chrome
