# general

## Environment Variables

This repository uses `direnv` to manage environment variables automatically.

### Initial Setup

After cloning the repository, enable direnv:

```console
direnv allow
```

## Terminal.app Configuration

- Change default profile to Basic
- Set font to Hack Nerd Font / Regular / 13
- Set background color to pitch black 100%
- Close window if shell exits cleanly
- Set Option key as Meta (Profiles → Keyboard → Use Option as Meta key)
